## Что такое branch prediction, в чем его идея?

**Branch Prediction** (предсказание ветвлений) — это критически важная оптимизация в современных процессорах, 
которая позволяет исполнять инструкции спекулятивно, ещё до того, как станет известно,
какая ветка кода будет выбрана.

* см. пример

## Как подсказать компилятору (средствами `C++20`, а также без него), какая из веток `if` более вероятна?

* Атрибуты `[[likely]]` и `[[unlikely]]`

```c++
int test(int x) {
    if (x > 0) [[likely]] {
        return 1;
    } else [[unlikely]] {
        return 0;
    }
}
```

## Как это отразится на ассемблерном коде?

```asm
; Без подсказок + O2
test:
    test    edi, edi
    jle     .L2
    mov     eax, 1 
    ret
.L2:
    xor     eax, eax
    ret
```

```asm
; С подсказкой + O2
test:
    test    edi, edi
    jg      .L4           
    xor     eax, eax      
    ret
.L4:
    mov     eax, 1
    ret
```
