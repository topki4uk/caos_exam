## Что такое `rlimit` для процесса?

* `RLIMIT` — это механизм в Unix/Linux системах для ограничения потребления ресурсов (памяти, CPU, файлов и т.д.) процессами.

## Как пользоваться функциями `getrlimit` и `setrlimit`?

```shell
  man 2 getrlimit
```

```shell
  man 2 setrlimit
```

* **Soft Limit** (текущий лимит) - что процесс может использовать сейчас
* **Hard Limit** (максимальный лимит) - абсолютный максимум, до которого можно поднять soft limit

# Что произойдет, если эти ограничения будут превышены?

| Ресурс         | При превышении Soft Limit    | При превышении Hard Limit   |
|----------------|------------------------------|-----------------------------|
| `RLIMIT_CPU`   | `SIGXCPU` (можно обработать) | `SIGKILL`                   |
| `RLIMIT_AS`    | `ENOMEM `                    | `ENOMEM `, `SIGKILL`        |
| `RLIMIT_STACK` | `SIGSEGV`                    | `SIGSEGV `, `SIGKILL`       |
