# Перенаправление ввода-вывода.

### Как в терминале направить вывод команды в файл с перезаписью файла?

```shell
  # Перезапись файла
  echo "Hello, world" > file.txt
```

```shell
  # Дозапись файла
  echo 123 > file.txt
  echo 456 >> file.txt
  echo 789 >> file.txt
```

### Как перенаправить вывод одного из потоков (cout или cerr) в файл или в другой поток?

```shell
  # Разветвление записи
  echo 123 | tee second.txt another.txt
```

### Как подавить вывод какого-то из потоков?

```shell
  # Поиск без подавления
  find / -name "*.core"  # очень много ошибок
```

```shell
  # Поиск с подавлением
  find / -name "*.core" 2> /dev/null  # перенаправили поток
```
