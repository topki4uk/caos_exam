## Что такое `fifo`-файлы? Как создать такой файл из терминала, а также программно?

```shell
  mkfifo mypipe
```

```shell
  man 3 mkfifo
```

## Что, если несколько процессов пишут в один и тот же `fifo`? 

* Разные процессы будут конкурировать за запись в поток, из-за этого мы теряем порядок сообщений

```shell
  # Запишем в fifo
  ./producer first &
  ./producer second &
  ./producer third &
```

```shell
  # Прочитаем из fifo
  ./consumer
```

## Что, если несколько процессов читают один и тот же `fifo`?

* Потоки будут конкурировать за чтение данных из `FIFO`
