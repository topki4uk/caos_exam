## Что такое `pipes`?

`Pipes` (каналы/конвейеры) — это механизм межпроцессного взаимодействия (IPC), 
который позволяет передавать данные между процессами в Unix/Linux системах. 
Это однонаправленный поток данных: что записано в один конец, можно прочитать из другого.

```shell
  man 2 pipe
```

## В какой ситуации возникает ошибка `Broken Pipe`?

* Это такая ситуация, в которой мы пытаемся писать в `pipe`, если у нас нет его читателей.
* По факту в результате этого генерируется сигнал `SIGPIPE`, который надо отловить

## Как реализовать аналог оператора `|` в bash на Си?

* bash-конвейер реализуется при помощи использования сисколов `pipe` + `fork` + `dup2`

```shell
  ./conveyor
```
