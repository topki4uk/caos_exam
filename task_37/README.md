## Что такое семафоры в Linux, для чего они нужны?

**Семафор (Semaphore)** — это примитив синхронизации для управления доступом к общим ресурсам в многопоточных и многопроцессных системах.

```shell
  man 2 semop
```

Семафор — это счётчик с двумя атомарными операциями:
* `wait()` (P-операция) — уменьшает счётчик, ждёт если он равен 0
* `signal()` (V-операция) — увеличивает счётчик

## Что позволяет делать сисколл `futex`?

* Он позволяет создавать эффективные примитивы синхронизации с минимальным переходом в ядро.

```shell
  man 2 futex
```
