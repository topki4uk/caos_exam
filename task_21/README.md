## Какие бывают права доступа к памяти?

* `PROT_READ` - чтение из памяти
* `PROT_WRITE` - запись в память
* `PROT_EXEC` - выполнение кода из памяти
* `PROT_NONE` - запрет любых операций

## Что означает ошибка Illegal intstruction?

**Illegal Instruction** - это сигнал, который процессор генерирует, когда пытается выполнить инструкцию 
(машинную команду), которую не может распознать или выполнить в текущем контексте.
