## Что такое процесс?

Процесс — это экземпляр выполняющейся программы. Он включает в себя:
- Виртуальное адресное пространство (maps)
- Таблицу файловых дескрипторов
- Контекст выполнения (регистры, стек, счетчик команд)
- Учетные данные (UID, GID)
- Состояние (running, sleeping, zombie и т.д.)
- Ресурсы (память, CPU, IO)

## Как посмотреть все процессы в системе?

```shell
  htop
```

## Что такое pid, ppid?

* **PID** - уникальный числовой идентификатор процесса в операционной системе.
* **PPID** - PID процесса-родителя, который создал данный процесс.

## Как посмотреть дерево процессов?

```shell
  # Через утилиту pstree
  pstree
```

```shell
  # Можно по конкретному процессу
  pstree -p $(pgrep firefox)
```

```shell
  # Через ps с флагом --forest
  ps -e --forest
```

```shell
  # htop + F5
  htop
```

## Что такое uid, euid и cwd данного процесса, как их узнать и как поменять?

* **UID (Real User ID)** - реальный идентификатор пользователя, который запустил процесс
* **EUID (Effective User ID)** - эффективный идентификатор пользователя, используемый для проверки прав доступа
* **CWD (Current Working Directory)** - текущая рабочая директория процесса

```shell
  # Можно узнать из директории /proc
  cat /proc/$(pgrep htop)/status
```

```text
// ls -l
// Есть такие вот исполняемые файлики

total 40
-rwxrwxr-x 1 alex alex 15968 Jan 15 11:56 default_example
-rwsrwxr-x 1 root alex 15968 Jan 15 11:56 super_example
```

```shell
  # Посмотрим на id наших пользователей
  cat /proc/$(pgrep default_example)/status
```

```text
TracerPid:      0
Uid:    1000    1000    1000    1000  # Запускал обычный пользовать - его права
Gid:    1000    1000    1000    1000
FDSize: 256
```

```shell
  # И для super варианта
  cat /proc/$(pgrep super_example)/status
```

```text
TracerPid:      0
Uid:    1000    0       0       0   # Запускал обычный пользователь - права рута
Gid:    1000    1000    1000    1000
FDSize: 256
```

### Для `cwd` аналогично

```shell
  # Можно узнать из директории /proc
  ls -la /proc/$(pgrep htop)/cwd
```
