## Расскажите про библиотеку `seccomp`.

* `seccomp` — это механизм безопасности ядра Linux, 
который позволяет процессу перейти в "безопасный" режим, 
где он может выполнять только ограниченный набор системных вызовов.

Она позволяет:
* Ограничивать системные вызовы, которые может выполнять процесс
* Фильтровать аргументы системных вызовов
* Работать на уровне ядра (очень эффективно)

## Покажите на примере, как запретить программе вызывать определенные сисколлы.

```shell
  # Если не установлен
  sudo apt-get install libseccomp-dev -y
```

```shell
  # Соберем все вместе
  g++ zapet_seccomp.cpp -lseccomp -o zapet_seccomp
```

##  Как получить ошибку `Bad system call (core dumped)`?

* Использовать запрещенный сискол при `seccomp`, это вызовет такую ошибку.
