# Блочные и символьные устройства.

### Приведите примеры того и другого.

* Блочный девайс. `/dev/sda1` - диск памяти
* Символьный девайс. `/dev/random`, `/dev/zero`, `/dev/null`

### Как прочитать данные с какого-нибудь символьного устройства, а также отправить данные на устройство?

Прочитать можно при помощи утилиты `cat`.

```shell
  cat /dev/random
```

Отправить можно при помощи команды `echo`:

```shell
  echo hello > /dev/null
```

### Что такое виртуальные устройства, что такое `/dev/null`, `/dev/random`, `/dev/zero`?

* `/dev/null` - устройство для игнора сообщений. Все, что туда попало - исчезнет.
* `/dev/random` - вывод случайных бит.
* `/dev/zero` - вывод ноликов.

```shell
  cat /dev/zero > file.txt
```

### Как сделать, чтобы приложение направило свой `stdout/stderr` на определенный терминал? 

```shell
  # Посмотреть номер девайса-терминала
  tty 
```

```shell
  # Отправить в другой терминал
  echo "hello from other terminal" > /dev/pts/4 
```

### Где в файловой системе хранится информация о CPU, об оперативной памяти, о жестких дисках / SSD?

```shell
  # Статы для CPU
  cat /proc/cpuinfo
```

```shell
  # Статы для RAM
  cat /proc/meminfo
```

```shell
  # Статы для HDD/SSD
  cat /proc/partitions
```

```shell
  # Статы для mounted разделов
  cat /proc/mounts
```
