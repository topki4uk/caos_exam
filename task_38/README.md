## Что такое ассемблер?

**Ассемблер** — низкоуровневый язык программирования, мнемоническое представление машинного кода.

## Какие есть разновидности ассемблера?

Для разных архитектур есть разные ассемблеры:
* `x86/x86-64 (Intel/AMD)`
  * `NASM` - самый популярный
  * `MASM` - для Windows
* `ARM`
  * `ARM Unified Assembler Language (UAL)` - современный
  * `AArch64` - для ARMv8 64-бит

## Что такое регистры?

* **Регистр** - это такая ячейка памяти, откуда процессор читает данные.
* Процессор читает память так: с помощью MMU (memory management unit) обращается к ОЗУ,
значение записывается в регистр.

## Перечислите основные регистры в архитектуре x86 и их предназначение.

#### 32-битные регистры:
* `EAX` (Extended Accumulator Register) - используется для арифмет. операций
* `EBX` (Extended Base Register) - указатель на данные (базовый)
* `ECX` (Extended Counter Register) - счётчик для циклов
* `EDX` (Extended Data Register) - данные для операций


* `EBP` (Extended Base Pointer) - указатель на базу стека (база фрейма)
* `ESP` (Extended Stack Pointer) - указатель вершины стека

#### 64-битные регистры (x86-64):
* `RAX`, `RBX`, `RCX`, `RDX` - их назначение описано выше
* `RBP`, `RSP` - их назначение описано выше

#### EFLAGS/RFLAGS (Flags Register)

* `ZF` (Zero) - результат ноль
* `PF` (Parity) - чётность
* `SF` (Sign) - знак (отрицательный)
* `IF` (Interrupt) - разрешение прерываний

#### Всего есть очень много регистров, но это самые базовые

## Расскажите про основные ассемблерные инструкции и их синтаксис: mov, арифметические инструкции, логические инструкции.

- `mov` — копирование из `src` в `dest`

```asm
mov dest, src   ; dest = src
```

- **Арифметика**: 
  - `add` - сложение

```asm
add eax, edx    ; eax += edx
```

  - `sub` - вычитание
  - `imul` - знаковое умножение (не теряем знак)
  - `idiv` - знаковое деление (не теряем знак)
- **Логические**: 
  - `and` - очев

```asm
and eax, edx    ; eax = eax & edx
```

  - `or` - очев
  - `xor` - очев
  - `not` - очев
  - `shl`/`shr` - логический битовый сдвиг
  - `sal`/`sar` - арифметический битовый сдвиг (очень важно для знаковых чисел)
